\b;Инструкция \c;pendown\n;
Синтаксис:
\s;\c;pendown ( color, width );\n;
Инструкция \c;pendown();\n; приказывает боту опустить кисть чтобы начать рисование.

\t;color: \c;\l;int\u cbot\int;\n; (значение по умолчанию: \c;Black\n;)
\l;Название цвета\u cbot\colors;.

\t;width: \c;\l;float\u cbot\float;\n; (default value: \c;0.5\n;)
Толщина кисти. Толщина не может быть больше \c;1.0\n; и не может быть меньше \c;0.0\n;. Passing a higher value will result in the width being the highest possible and passing a lower value will result in the width being the lowest possible.

\t;Возвращаемое значение: \c;\l;int\u cbot\int;\n;
Обычно при возникновении ошибки выполнение программы останавливается. Избежать этого можно применив инструкцию \c;\l;errmode\u cbot\errmode;(0)\n; . Тогда при возникновении ошибки будет возвращено значение отличное от нуля инструкцией \c;pendown();\n;.
\c;== 0 \n;Кисть подогтовлена к рисованию
\c;!= 0 \n;Инструкцию не удалось выполнить
