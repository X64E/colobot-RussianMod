\b;Константы\n;

Константы — это фиксированные значения, которые не могут изменяться программой. Они могут быть любых типов.

В язык CBOT по умолчанию заложены:

Костанты для обозначения категорий объектов.

Константы для обозначения цвета.

Также есть специальные константы для некоторых операций.
