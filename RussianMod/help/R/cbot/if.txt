\b;Оператор \c;if\n; и \c;else\n;
Оператор \c;if() {}\n; выполняет набор инструкций, только в случае если выполняется определенное условие. Условие должно быть изложено в скобках \c;()\n;, а код который должен быть выполнен должен быть указан в фигурных скобках \c;{}\n;.

\b;Применение
Вот вам конкретный пример: Бот станет стрелять, только если цель находится ближе чем в 40 метрах:
\c;
\s;	item = \l;radar\u cbot\radar;(AlienAnt);
\s;	if (\l;distance\u cbot\dist;(position, item.position) < 40)
\s;	{
\s;		fire(1);
\s;	}
\n;
Также можно проверить, существование объекта. Если инструкция \c;\l;radar\u cbot\radar;();\n; не находит искомый объект, она возвращает значение \c;null\n;. Исходя из этого можно проверить наличие объекта с помощью условия \c;(item == null)\n;, или с помощью \c;(item != null)\n;. Два знака равенства \c;==\n; проверяют равность, знак восклицания, за которым идет знак равенства \c;!=\n; проверяют неравенство. Вот тестовая программа, которая отправит бот на перезарядку \l;энергобатареи\u object\power; только в случае наличия \l;электростанции\u object\station;:
\c;
\s;	item = \l;radar\u cbot\radar;(PowerStation); // Ищем Электростанцию с помщью радара
\s;	if (item != null)           // Если электростанция обнаружена
\s;	{
\s;		\l;goto\u cbot\goto;(item.position);    // Перемещаемся к электростанции
\s;		\l;wait\u cbot\wait;(5);                // Подзаряжаем аккумулятор в течении 5 сек.
\s;	}
\n;
\b;Для специалистов
Синтаксис:
\s;\c;if ( \l;условие\u cbot\cond; )
\s;{
\s;	\n;\l;Блок\u cbot\bloc; инструкций A ...\c;
\s;}
\s;else
\s;{
\s;	\n;\l;Блок\u cbot\bloc; инструкций Б ...\c;
\s;}
\n;
Данной условной структурой вы можете выполнить \l;блок\u cbot\bloc; A или \l;блок\u cbot\bloc; Б в зависимости от \l;условия\u cbot\cond;. Если условие выполняется, будут выполнены инструкции блока A. Если условие не выполняется, то будут выполнены инструкции блока Б.
Part \c;else { }\n; is not compulsory.

\t;Краткое написание if
Если после if следует только одна инструкция можно не использовать фигурные скобки.

\s;\c;if ( a>3 ) a=a+1;

тоже самое, что:

\s;\c;if ( a>3 )
\s;{
\s;	\n;a=a+1;\c;
\s;}


\t;Внимание
Не следует ставить \l;точку с запятой\u cbot\term; после \c;if( )\n;.
