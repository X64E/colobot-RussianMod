\b;Инструкция \c;public\n; (для специалистов)
Эта команда имеет две различные цели:

1) Сделать функцию доступной для других ботов.
2) Сделать член класса доступным извне для других классов.

\b;Инструкция \c;public\n; для функций
Если вы укажите ключевое слово \c;public\n; перед определением \l;функциии\u cbot\function;, то данная функция будет доступна для других ботов.

Например, если первый бот содержит в себе публичную функцию:
\c;
\s;public void object::Segment(float dist, float angle)
\s;{
\s;	move(dist);
\s;	turn(angle);
\s;}
\n;
Тогда мы можем применить это функцию в программе второго бота:
\c;
\s;extern void object::Square( )
\s;{
\s;	for ( int i=0 ; i<4 ; i++ )
\s;	{
\s;		Segment(10, 90);
\s;		Fire(1);
\s;	}
\s;}
\n;
Например сниффер может сканировать землю, но не может строить здания. А робот строитель может строить здания, но не может сканировать землю на наличие ресурсов. При нахождении залежей ресурсов сниферром, вы можете отправить роботу строителю задание на постройку здания. Используя публичную функцию в коде строителя.

Если вы объявили функцию \c;общедоступной\n;, то вы сможете использовать ее в других миссиях.

Если бот, содержащий \c;общедоступную\n; функцию будет уничтожен, то другие боты, использовавшие её будут остановлены с ошибкой.

\b;Инструкция \c;public\n; для классов
\c;public\n; is also an access modifier for \l;class\u cbot\class; members, which is the default one. Public members can be accessed from outside of the class definition.
\c;
\s;public class MyClass
\s;{
\s;	int b;                     // общедоступное поле по-умолчанию
\s;	public  int a;             // общедоступное поле
\s;	private point position;    // приватное поле (доступно внутри класса)
\s;}
\s;void Test()
\s;{
\s;	MyClass item;
\s;	item.a = item.b = 12;      // эти поля общедоступны их можно изменять
\s;	message( item.position );  // приведет к ошибке, т.к. это поле приватное
\s;}
\n;
